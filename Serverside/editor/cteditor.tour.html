Add a new Tour

To create a new tour, type its name into the Title box. You can also put an optional description in the box labeled Description. Click the Add button to save this tour and add it to the list of tours.

To go back to the main page, click the Back button in the top left corner of the screen.

List Tour Panoramas

Back

To add a panorama to this tour, click on the drop down menu to see a list of all the current existing panoramas. Select the panorama and click the Add button to add that panorama to the end of the list. Repeat this until you have the full list of panoramas that you want for this tour. To delete a panorama from the tour, click the x on the right side of that panorama in the list.

To go back to the list of tours, click the Back button in the top left corner of the screen.
